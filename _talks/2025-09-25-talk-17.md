In this talk, the recently released tool Continuous CyberBattleSim—an extension of Microsoft CyberBattleSim—was presented at an internal SuperViZ project meeting. This tool integrates an embedding model to support continuous spaces for reinforcement learning tasks, features an extended scenario generation pipeline, and provides a unified framework for RL training and evaluation.
